# Sprint 8. Итоговый проект — Задание 1: юнит-тесты

## Описание
Автоматизация юнит тестирования программы для заказа бургеров Stellar Burgers. Проект покрывает тестами ключевые классы приложения с использованием моков и параметризации.

## Тестовые сценарии

### 1. Тестирование класса Bun
- Проверка получения названия булки
- Проверка получения цены булки
### 2. Тестирование класса Burger
- Проверка установки булки для бургера
- Проверка добавления ингредиента в бургер
- Проверка удаления ингредиента из бургера
- Проверка перемещения ингредиента в списке
- Проверка расчета стоимости бургера (булка + все ингредиенты)
- Проверка формирования корректного чека заказа
### 3. Тестирование класса Database
- Проверка получения доступных булок из базы данных 
- Проверка получения доступных ингредиентов из базы данных
### 4. Тестирование класса Ingredient
- Проверка получения цены ингредиента
- Проверка получения названия ингредиента
- Проверка получения типа ингредиента (соус/начинка)

## Запуск тестов

1. Клонировать репозиторий
```bash
git clone https://github.com/KBulat/Diplom_1.git
```

2. Установить зависимости
```bash
pip install -r requirements.txt
```

3. Запустить тесты
```bash
pytest -v
```
